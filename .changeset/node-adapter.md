Add Node.js adapter (`@sanity-labs/logo-soup/node`) for server-side logo measurement using `@napi-rs/canvas`. Extract shared pixel math into `measureContent` pipeline used by both browser and Node paths. Includes `measureImage`, `measureImages`, and re-exports of `createNormalizedLogo`, `calculateNormalizedDimensions`, and `getVisualCenterTransform` with all supporting types.
